# Fix Broken Styling Guide

If the imported page looks "plain," has the wrong fonts, or missing colors, follow these 3 steps to fix it.

## 1. Fix Fonts (Critical)

The design relies on specific Google Fonts that are likely missing from your app.

**Action:** Add this line to your `index.html` file inside the `<head>` tag:

```html
<link href="https://fonts.googleapis.com/css2?family=DM+Serif+Display:ital@0;1&family=Outfit:wght@100..900&display=swap" rel="stylesheet">
```

## 2. Add CSS Variables

If your application uses **Tailwind CSS v3** (most common), it cannot read the new v4 `@theme` syntax. You must manually add these variables.

**Action:** Copy these variables into your global CSS file (e.g., `globals.css`, `index.css`, or `App.css`) inside the `:root` block:

```css
@layer base {
  :root {
    --font-sans: 'Outfit', sans-serif;
    --font-heading: 'Outfit', sans-serif;
    --font-serif: 'DM Serif Display', serif;
    --font-code: 'Courier New', monospace;

    /* Dark Theme Colors */
    --background: 240 10% 4%;
    --foreground: 0 0% 98%;

    --primary: 217 100% 60%;
    --primary-foreground: 0 0% 100%;

    --secondary: 240 4% 16%;
    --secondary-foreground: 0 0% 98%;

    --muted: 240 4% 16%;
    --muted-foreground: 240 5% 65%;

    --accent: 45 93% 47%;
    --accent-foreground: 240 10% 4%;

    --border: 240 4% 16%;
    --input: 240 4% 16%;
    --ring: 45 93% 47%;

    --radius: 0.5rem;
  }
}

/* Ensure these base styles are applied */
@layer base {
  * {
    @apply border-border;
  }
  body {
    @apply font-sans antialiased bg-background text-foreground;
  }
  h1, h2, h3, h4, h5, h6 {
    @apply font-heading;
  }
}
```

## 3. Update Tailwind Config (For Tailwind v3 Users)

If you are using Tailwind v3, you need to tell it to use the variables you just added.

**Action:** Update your `tailwind.config.js` file to include this `extend` block:

```javascript
/** @type {import('tailwindcss').Config} */
module.exports = {
  // ... other config
  theme: {
    extend: {
      colors: {
        border: "hsl(var(--border))",
        input: "hsl(var(--input))",
        ring: "hsl(var(--ring))",
        background: "hsl(var(--background))",
        foreground: "hsl(var(--foreground))",
        primary: {
          DEFAULT: "hsl(var(--primary))",
          foreground: "hsl(var(--primary-foreground))",
        },
        secondary: {
          DEFAULT: "hsl(var(--secondary))",
          foreground: "hsl(var(--secondary-foreground))",
        },
        destructive: {
          DEFAULT: "hsl(var(--destructive))",
          foreground: "hsl(var(--destructive-foreground))",
        },
        muted: {
          DEFAULT: "hsl(var(--muted))",
          foreground: "hsl(var(--muted-foreground))",
        },
        accent: {
          DEFAULT: "hsl(var(--accent))",
          foreground: "hsl(var(--accent-foreground))",
        },
      },
      fontFamily: {
        sans: ["Outfit", "sans-serif"],
        heading: ["Outfit", "sans-serif"],
        serif: ["DM Serif Display", "serif"],
        code: ["Courier New", "monospace"],
      },
    },
  },
}
```
